Detailed Description

The HTTPManager class is used to contact the server, but not to send files.

The HTTPManager class manages all the connections with the server. It can call servlets on the server to che ids and parameters. This class do NOT send files: it is only used to make and HTTP request and manage the answears from the server. Usually, the server respond with a JSON string of data.
