Detailed Description

The CameraModule class is used to manage image acquisition and manipulation.

The CameraModule class is the only Module who can take, save and manipulate photos. It use an external program build with OpenCV to performa a simple crop based on local parameters.

The object of this class is registered in log with the name "Camera Module", where no other names are provided.
